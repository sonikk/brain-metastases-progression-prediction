# Прогнозирование риска интракраниальной прогрессии метастазов в головном мозге после радиохирургии на аппарате «Гамма-нож»

ML-проект по прогнозированию риска интракраниальной прогрессии (ИП) у пациентов с метастазами в головном мозге после радиохирургического лечения на аппарате «Гамма-нож».

Проект основан на обезличенных клинических данных **916 пациентов**, проходивших лечение в период с 2005 по 2017 год, и направлен на поддержку персонализированного подхода к постлучевой терапии и принятию врачебных решений.

---

# Бизнес-задача

После проведения радиохирургии у пациентов может развиваться **ИП**:

- локальный рецидив
- появление новых метастазов
- рост опухоли в зоне облучения

Раннее прогнозирование риска прогрессии позволяет:

- своевременно выявлять пациентов группы высокого риска
- персонализировать последующее лечение
- повышать эффективность терапии
- улучшать качество жизни пациентов
- поддерживать принятие врачебных решений

Классические прогностические системы (RPA, GPA) используют ограниченный набор факторов и не позволяют учитывать весь объем современной клинической информации. Методы машинного обучения позволяют строить более точные и интерпретируемые прогностические модели.

---

# Данные

## Источник

Обезличенные клинические данные из Единого регистра пациентов, леченных радиохирургией на аппарате «Гамма-нож».

### Период наблюдения

**июнь 2005 — январь 2017**

### Объем выборки

**916 пациентов**

### Возраст

**24–84 года**

### Первичные диагнозы

- немелкоклеточный рак легкого
- рак молочной железы
- меланома
- рак почки
- колоректальный рак

### Признаки

- возраст
- пол
- первичный онкологический диагноз
- наличие экстракраниальных метастазов
- число очагов в головном мозге
- количество радиохирургических процедур
- объем очагов
- индекс Карновского
- лекарственное лечение
- время до метастазирования
- время реагирования
- наличие операций
- облучение всего головного мозга (ОВГМ)

### Целевая переменная

**Бинарный признак интракраниальной прогрессии**

- 0 — нет прогрессии
- 1 — есть прогрессия

# Этапы работы

## 1. Предобработка данных

- обработка пропусков
- нормализация числовых признаков (Min-Max Scaling)
- one-hot encoding категориальных признаков
- формирование целевой переменной
- feature engineering

## 2. EDA

- исследовательский анализ данных
- визуализация распределений признаков
- корреляционный анализ
- сравнение групп с прогрессией и без прогрессии
- анализ числовых и категориальных признаков

## 3. Сравнение моделей

Было протестировано **11 моделей машинного обучения**:

- Logistic Regression
- Decision Tree
- kNN
- SVM
- LDA
- Random Forest
- AdaBoost
- Bagging
- Voting Classifier
- Stacking
- CatBoost

Оценка качества проводилась с помощью:

- Stratified 5-Fold Cross Validation
- метрики **Balanced Accuracy**

## 4. Финальная модель

Для итоговой модели был выбран:

# CatBoostClassifier

Дополнительно использовались:

- RandomizedSearchCV для подбора гиперпараметров
- калибровка порога классификации через OOF (Out-of-Fold prediction)
- SHAP-анализ для интерпретации модели

---

# Результаты

## Лучшая модель

**CatBoostClassifier**

## Метрика качества

**Balanced Accuracy = 0.7839**

## Оптимальный порог классификации

**0.525**

## Наиболее важные признаки

По результатам SHAP-анализа:

1. Количество радиохирургических процедур
2. Количество очагов в головном мозге
3. Возраст
4. Время реагирования
5. Индекс Карновского

Модель показала высокую точность стратификации риска и может использоваться для поддержки принятия клинических решений.
